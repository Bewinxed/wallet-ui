# Wallet UI Monorepo

Work in progress.

## Getting started

```shell
git clone https://github.com/wallet-ui/wallet-ui.git
cd wallet-ui
pnpm install
```

## Development

### Build

```shell
pnpm run build
```

### Lint

Run the linter:

```shell
pnpm run lint
```

Fix any linting issues:

```shell
pnpm run style:fix
```

### Test

```shell
pnpm run test
```
